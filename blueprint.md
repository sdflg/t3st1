# 대한민국 연금복권 번호 추첨 사이트

## 개요

이 프로젝트는 대한민국 연금복권 번호를 무작위로 추첨하는 웹 애플리케이션입니다. 사용자는 버튼을 클릭하여 새로운 복권 번호를 생성하고 확인할 수 있습니다.

## 디자인 및 기능

### 스타일 및 디자인
- **글꼴**: 시스템 기본 글꼴을 사용하여 가독성을 높입니다.
- **색상**: 전체적으로 어두운 테마를 사용하여 눈의 피로를 줄이고, 번호가 강조될 수 있도록 밝은 색상을 사용합니다.
- **레이아웃**: 중앙 정렬된 레이아웃으로 사용자가 콘텐츠에 집중할 수 있도록 합니다.
- **애니메이션**: 번호가 추첨될 때 부드러운 애니메이션 효과를 추가하여 사용자 경험을 향상시킵니다.

### 기능
- **번호 추첨**: '번호 추첨' 버튼을 클릭하면 1조부터 5조까지, 그리고 000000부터 999999까지의 숫자 중에서 무작위로 번호를 생성합니다.
- **결과 표시**: 추첨된 번호는 화면 중앙에 큰 글씨로 표시됩니다.

## 현재 구현 계획

1.  **HTML (`index.html`)**:
    *   사이트 제목, 번호가 표시될 영역, 번호 추첨 버튼을 포함하는 기본 구조를 설정합니다.
2.  **CSS (`style.css`)**:
    *   어두운 테마, 중앙 정렬 레이아웃, 버튼 및 번호 표시 영역의 스타일을 지정합니다.
3.  **JavaScript (`main.js`)**:
    *   '번호 추첨' 버튼 클릭 시 연금복권 번호를 생성하고 화면에 표시하는 로직을 구현합니다.
